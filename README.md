## ⚙️ Arquitetura de Dados

- Dados salvos em **LocalStorage** para acesso offline imediato.  
- Sincronização com **Firestore** em segundo plano.  
- Indicadores de status:  
  - 🟡 Sincronizando  
  - 🟢 Conectado  
  - 🔴 Offline  

---

## 🔧 Funcionalidades

- **Dashboard**: visão geral de tarefas e alertas  
- **Tarefas**: criar, editar, excluir, filtrar  
- **Colaboradores**: cadastro de pessoas vinculadas a setores  
- **Setores**: criação com cor e ícone personalizados  
- **Relatórios**: exportação CSV, impressão e gráficos  

---

## 💰 Custo do Firebase

Plano Blaze (com tier gratuito):  
- Leituras: até 50.000/dia  
- Gravações: até 20.000/dia  
- Armazenamento: até 1 GB  
- Uso estimado: dentro do limite gratuito
